# OneDayTrading 네 번째 페이지 워크플로우

## 목적

캐러셀을 읽은 독자가 인스타그램 계정을 팔로우하고 `onedaytrading.net`에서 더 많은 뉴스·종목 자료를 확인하도록 유도하는 브랜드 엔딩 페이지다.

4페이지는 중앙 집중 흐름을 유지하되 딱딱한 영화 크레딧 문법을 그대로 쓰지 않는다. 1페이지의 색감과 글 스타일을 가져온 부드러운 현대적 브랜드 엔딩으로 구성한다.

## 입력 데이터

```json
{
  "instagram_handle": "@onedaytrading.io",
  "website": "onedaytrading.net",
  "market_label": "해외주식",
  "logo": "NEWLOGO.png",
  "background_image": ""
}
```

## 메시지 구조

```text
브랜드 선언: 오늘의 뉴스가 내일의 판단이 되도록.
가치 제안: 놓치기 쉬운 시장의 맥락을 한 장씩 더 선명하게 연결합니다.
CTA 1: 인스타그램 팔로우
CTA 2: 더 많은 증시 분석·종목 자료 확인
```

## 카피 규칙

- 브랜드 선언: 최대 2줄, 12~22자
- 가치 제안: 최대 2줄
- 인스타 CTA: `매일 핵심 뉴스 받기 →`
- 웹 CTA: `더 많은 분석·종목 자료 보기 →`
- `팔로우 부탁드립니다`처럼 수동적인 표현보다 독자가 얻는 이점을 먼저 말한다.
- 과도한 느낌표, `무조건`, `급등주`, `수익 보장` 같은 표현은 금지한다.

## 디자인 규칙

- 1~3페이지와 색감만 연결하고 기사 배경 사진은 완전히 제거한다.
- 1페이지와 같은 짙은 남색, 절제된 적색, 짧은 하늘색 포인트를 사용한다.
- 청색과 적색은 부드러운 글로우로 확산시켜 배경이 평면적으로 보이지 않게 한다.
- 로고 → 브랜드 선언 → 인스타그램 → 웹사이트 → 엔드 크레딧이 하나의 중앙 세로축을 따라 흐르게 한다.
- 좌우 비대칭 배치는 사용하지 않고 모든 핵심 텍스트를 중앙 정렬한다.
- 큰 제목은 `Noto Sans KR Black`, 영문 라벨은 파란색 대문자로 표현한다.
- 대문자 크레딧 문구와 긴 중앙 신호선은 사용하지 않는다.
- 인스타그램 영역은 별도 패널 없이 계정명과 둥근 적색 CTA만으로 표현한다.
- 제목은 1페이지와 같은 `Noto Sans KR Black`, 설명은 `Noto Sans KR`을 사용한다.
- 짧은 하늘색 구분선, 넓은 행간, 소문자 영문 라벨을 고정 시각 자산으로 사용한다.
- 최종 간결형에서는 패널, 흰색 테두리, 하단 구분선, 장식용 영문 라벨을 제거한다.
- 중앙 콘텐츠는 `브랜드 메시지 → 인스타 계정/팔로우 → 사이트/추가 자료` 순서로 밀집 배치한다.
- 필요한 정보 이외의 슬로건 반복과 장식 문구는 넣지 않는다.
- 하단 반복 슬로건과 페이지 번호는 표시하지 않는다.
- 로고, 제목, 인스타 CTA, 사이트 CTA의 중앙 X축을 정확히 통일한다.
- 전체 콘텐츠 그룹은 캔버스 상단 기준 배치 후 `Y +85px` 이동해 4:5 화면의 시각적 중앙에 맞춘다.

## 최종 확정 구성

```text
NEWLOGO
onedaytrading.

오늘의 뉴스가
내일의 판단이 되도록.

시장에 흩어진 신호를 읽기 쉬운 맥락으로 정리합니다.

매일 핵심 뉴스 받기
@instagram_handle
[팔로우하기 +]

더 많은 뉴스와 자료
onedaytrading.net
증시 분석 · 종목별 뉴스 · 시장 자료
사이트에서 더 보기 ↗
```

최종본에서는 다음 요소를 사용하지 않는다.

- 흰색 테두리
- 반투명 정보 박스
- 하단 구분선
- 반복 슬로건
- 페이지 번호
- 장식 목적의 긴 영문 문구

- 링크 텍스트는 이미지에 정확히 후처리하며 이미지 생성 모델에 맡기지 않는다.

## 자동 검수

- 인스타 계정명이 실제 계정과 일치하는가
- `onedaytrading.net` 철자가 정확한가
- 인스타 팔로우와 웹사이트 방문의 이점이 각각 명확한가
- CTA가 두 개를 초과하지 않는가
- 1~3페이지의 색상·폰트와 자연스럽게 연결되는가
- 이미지에 투자 수익을 보장하는 표현이 없는가
