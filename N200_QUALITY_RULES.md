# N200 최종 품질 규칙

## 1. 1페이지 훅 카드

- 기사 대표 이미지는 사실 확인 참고용이다. 이미지 안에 원문 글자·로고·카드 UI가 있으면 배경으로 직접 사용하지 않는다.
- 배경은 텍스트 없는 N200 오리지널 비주얼로 생성한다. 제목 영역에는 충분한 여백을 남긴다.
- 제목은 카드용 두 줄 이내로 재작성한다. 원문 제목을 그대로 축소하지 않는다.
- 부제는 한 문장, 두 줄 이내로 편집한다. 잘림·말줄임·불완전 문장을 허용하지 않는다.

## 2. 2페이지 영상 카드

- 사건 직접 증거 영상을 최우선으로 찾는다.
- 직접 증거가 없을 때만 인물 컨텍스트 영상을 사용한다.
- 인물 컨텍스트는 단순히 이름이 자막에 나오는 장면을 사용하지 않는다. 본인 자기소개·본인 발언 신호를 우선하며, 진행자 소개·질문 장면은 탈락시킨다.
- 최신 영상이 우선이다. 전체 메타데이터에서 게시일을 다시 확인하며, 최근 12개월을 기본 창으로 쓴다. 날짜가 없는 검색 스니펫은 최신 영상과 동등하게 취급하지 않는다.
- 화면 인물 확인이 불완전하면 자동 게시하지 않고 `needs_clip_review`로 멈춘다.
- 외부 영상은 오른쪽 하단에 `(c) 채널명`을 표시하고, 출처·구간·권리 상태를 저장한다.
- 하단 본문은 면책 문장이 아니라 기사로 이어지는 `HOT ISSUE` 문장으로 쓴다. 영상 역할은 상단 라벨로만 구분한다.

## 3. 1~5장 완성 조건

1. 훅 이미지 — 오리지널 텍스트 없는 배경 + 짧은 제목·부제
2. 영상 — 최신성·인물·출처 검수 통과
3. 분석 이미지 — 원문에서 확인되는 영향 경로
4. 판단 이미지 — 확인 사실·미확인 변수 분리
5. 마무리 이미지 — N200 CTA

모든 장은 1080×1350, 동일한 다크 네이비 N200 팔레트, 통일된 로고·쪽수·푸터를 사용한다.
