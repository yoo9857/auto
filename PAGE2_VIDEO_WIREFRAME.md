# 페이지 2: 영상 증거 + 우리 해석

페이지 2는 긴 분석 본문이 아니라, 원본 영상을 근거로 우리만의 판단을 전달하는 고정 포맷이다.

```text
0–6%    EVIDENCE / 페이지 번호
7–48%   원본 영상(7–12초), 출처 라벨, 재생 표식
51–66%  OUR TAKE + 한 줄 질문형 제목
70–76%  영상 속 사실
79–86%  우리의 해석(파란 바)
89–100% 브랜드 풋터 / 02·04
```

원본 영상은 위 936×552px 영역에 `cover` 방식으로 배치한다. 출처 라벨은 항상 남기고, 영상 자체의 주장과 우리 해석을 분리한다. 영상 파일이 들어오면 플레이스홀더만 교체하며, 카피의 위치와 크기는 바꾸지 않는다.

와이어프레임 생성:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\render_page2_evidence.ps1
```
