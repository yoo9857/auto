# N200 링크 1개 완성 워크플로우

실행 명령:

```powershell
.\build_n200_full.ps1 -Url "https://onedaytrading.net/..."
```

생성 순서:

1. 기사 원문과 대표 이미지를 저장한다.
2. 사건 증거 영상을 우선 선택하고, 없으면 기사 속 인물의 연설·인터뷰·총회 장면을 컨텍스트 영상으로 찾는다.
3. 긴 영상은 자막에서 인물·주제·형식 키워드를 찾아 12초만 추출한다.
4. 1 훅 이미지, 2 영상, 3 분석 이미지, 4 판단 이미지, 5 마무리 이미지를 렌더링한다.
5. `DB\날짜-제목`에 정확히 5개 게시용 파일만 복제한다.

외부 이미지·영상은 자동으로 `review_required` 상태가 된다. 영상 출처는 화면 오른쪽 하단에 `© 채널명`으로 표시하며, 표기 자체는 재사용 허가를 대신하지 않는다.
